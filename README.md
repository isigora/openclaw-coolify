## üöÄ Features

### 1. Multi-Agent Sandbox Architecture
The main `bot` container acts as the Orchestrator. It can:
-   **Create Workers**: `create_sandbox.sh --stack nextjs` -> Spawns a Bun/Next.js container.
-   **Tunnel Access**: Assigns a `https://ProjectName.trycloudflare.com` URL automatically.
-   **Persist Data**: All state managed in `sandboxes.db` (SQLite).

### 2. Advanced Web Tools (`skills/web-utils`)
Stop worrying about bot detection. The platform includes a unified 5-stage scraper:
```bash
# Intelligent Auto-Mode (Retries all strategies)
skills/web-utils/scripts/scrape.sh --mode auto "https://blocked-site.com"
```

### 3. Integrated Services
The `docker-compose.yaml` template includes the essentials:
-   **Moltbot/OpenClaw**: The brain.
-   **SearXNG**: Private, tracking-free web search (JSON enabled).
-   *(Optional via Dashboard)*: Qdrant, Miniflux, Linkding.

### 4. Smart Deployment
*   **Cloudflare Tunnels**: Just set `CF_TUNNEL_TOKEN` for instant public URLs.
*   **Vercel + GitHub**:
    *   Auto-creates **Private Repos**.
    *   Refactors local DBs (SQLite) to **Vercel KV/Postgres** automatically.
    *   Verifies build (self-repair) before pushing.
    *   Syncs `.env` secrets.
